🚀 הגדרה ראשונית — 5 שלבים

1
פתח Apps Script חדש
היכנס ל-Apps Script עם חשבון Google → לחץ "New project"
🔗 פתח script.google.com ↗
2
הדבק את Code.gs
מחק את הקוד הקיים → הדבק → שמור (Ctrl+S).
👁 צפה בקוד ↗
3
הוסף קובץ index.html
ב-Apps Script: ➕ → HTML → שם: index (חובה!) → מחק הכל → הדבק.
4
הפעל fullSetup() — יוצר הכל
בחר fullSetup מהרשימה → לחץ ▶ Run → אשר הרשאות. בלוג תראה ✅ + URL של ה-Sheet.
5
פרסם ← הדבק את ה-URL למעלה
Deploy → New Deployment → Web App
• Execute as: Me
• Who has access: Anyone
→ Deploy → העתק URL → הדבק בשדה למעלה.

💡 יש דרך פשוטה יותר — ללא Apps Script

כל הסטאק הנוכחי חינמי לחלוטין. אם רוצה מינימום תחזוקה — יש חלופה: Google Sheet מפורסם כ-CSV.

🔵 שיטה נוכחית (Apps Script)

  • ✅ אבטחה מלאה (token)
  • ✅ פורטל HTML מעוצב
  • ✅ רענון חי כל 5 דקות
  • ✅ פילטור לפי לקוח
  • ⚠️ דורש Deploy חד-פעמי

🟢 חלופה פשוטה (CSV ציבורי)

  • ✅ אפס קוד לפרוס
  • ✅ עדכון מיידי — ערוך Sheet → נראה
  • ✅ אין Deploy אף פעם
  • ⚠️ כל מי שיש לו URL רואה הכל
  • ⚠️ אין token / פילטור
